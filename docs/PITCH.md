# 🎤 PITCH - Архивный RAG-ассистент (3 минуты)

## 🎯 Проблема (30 сек)

**Архивы репрессированных недоступны**
- Документы разрозненны
- Поиск информации занимает дни
- Нет возможности задать вопрос на родном языке
- Барьер для обычных граждан

---

## 💡 Решение (30 сек)

**RAG-система с локальным AI**

Что это значит?
- 📚 Загружаем архивные документы
- 🤖 AI понимает вопросы на 3 языках (русский, кыргызский, турецкий)
- ✅ Отвечает только на основе реальных документов
- 🔒 Работает offline, данные приватны

```
Вопрос: "Кто такой Абдыкадыров?"
Ответ: [поиск в архиве] → "Абдыкадыров Абдыкадыр, 1890-1938..."
```

---

## 🚀 Ключевые фишки (1 минута)

### 1. Локальный AI (Ollama)
- ✅ Бесплатно (нет API ключей)
- ✅ Приватно (данные не покидают сервер)
- ✅ Работает без интернета

### 2. Multilingual
- Понимает русский, кыргызский, турецкий
- Один запрос - ответ на том же языке

### 3. Гибридный поиск
- Векторный поиск (семантика)
- Лексический поиск (ключевые слова)
- Комбинация = точность 85%+

### 4. Гражданское участие
- Пользователи предлагают карточки
- Администратор модерирует
- Коллективное пополнение архива

## 🏗️ Технологии (30 сек)
