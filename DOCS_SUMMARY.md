## 🚀 Quick Start (что делать ПРЯМО СЕЙЧАС)

### Шаг 1: Прочитать (20 мин)
```bash
cat READY_FOR_HACKATHON.md   # Текущий статус
cat PITCH.md                  # ЗАПОМНИТЬ!
cat CHECKLIST.md              # Что делать дальше
```

### Шаг 2: Запустить систему (5 мин)
```bash
# Проверить Ollama
systemctl status ollama

# Запустить Docker
docker-compose up -d

# Открыть браузер
xdg-open http://localhost:8080
xdg-open http://localhost:8501
```

### Шаг 3: Протестировать (5 мин)
- Открыть RAG чат
- Задать вопрос: "Кто такой Абдыкадыров?"
- Убедиться что ответ приходит за 2-5 секунд
