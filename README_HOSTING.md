## 🚨 Частые вопросы

### Q: Сколько пользователей может обслуживать?
**A:** На VPS с 16GB RAM и llama3:8b:
- 10-20 одновременных пользователей
- 100+ пользователей в день (не одновременно)

С GPU - в 5-10 раз больше.

### Q: Можно ли использовать бесплатный хостинг?
**A:** Нет, Ollama требует минимум 8GB RAM. Бесплатные провайдеры дают максимум 1-2GB.

### Q: Нужен ли GPU?
**A:** Нет, но с GPU будет в 5-10 раз быстрее. 
- CPU: 2-5 сек на ответ
- GPU: 0.5-1 сек на ответ

### Q: Какая модель лучше?
**A:** Зависит от RAM:
- 8GB RAM → phi:2.7b (быстрая, но менее точная)
- 16GB RAM → llama3:8b (баланс скорости и качества)
- 32GB+ RAM → llama3:70b (лучшее качество, медленнее)

### Q: Как обновить проект?
**A:**
```bash
cd /srv/hackathon
git pull
docker-compose build
docker-compose up -d
```

### Q: Ollama не работает, что делать?
**A:** См. раздел "Проблемы и решения" в **DEPLOYMENT_WITH_OLLAMA.md**

## 🔐 Безопасность

3. ✅ Настройте firewall (ufw)
