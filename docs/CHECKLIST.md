## 🚨 Troubleshooting - если что-то не работает

### Ollama не отвечает
```bash
# Проверить статус
systemctl status ollama

# Перезапустить
sudo systemctl restart ollama

# Проверить что модель загружена
ollama list
ollama pull llama3:8b
```

### Docker контейнеры не стартуют
```bash
# Посмотреть логи
docker-compose logs backend_python
docker-compose logs postgres

# Пересоздать
docker-compose down
docker-compose up -d
```

### RAG не отвечает или долго думает
- Проверить что Ollama запущен
- Первый запрос всегда медленнее (загрузка модели)
- Проверить логи: `docker-compose logs backend_python | tail -50`

### База данных пустая
```bash
# Проверить что volume существует
docker volume ls | grep hackathon

# Запустить миграции
docker-compose exec backend_python alembic upgrade head
```

### Frontend не открывается
- Проверить что Nginx запущен: `docker-compose ps nginx`
- Проверить порты: `netstat -tlnp | grep 8080`
- Проверить логи: `docker-compose logs nginx`

## 📊 Метрики для презентации

### Производительность
- Скорость ответа RAG: 2-5 секунд (CPU) или 0.5-1 сек (GPU)
- Throughput: 20 одновременных пользователей
- Размер модели: 4.7GB (llama3:8b)
- Размер базы: ~100MB для 1000 карточек

### Стоимость
- VPS хостинг: $13-25/месяц (Contabo, Hetzner)
- GPU хостинг: $144-245/месяц (RunPod, Vast.ai)
- Development: бесплатно (локальный Ollama)

### Технические
- Языки: Python, C++, JavaScript
- Frameworks: FastAPI, Streamlit, LangChain
- Databases: PostgreSQL, ChromaDB
- AI: Ollama (llama3:8b)
- Deploy: Docker, Nginx
